導入方法

このガイドはWindows画面ですが、MacでもWindowsでも基本的には同じです。

1.Chromeを起動し、拡張機能設定に行きます。
右上のハンバーガーメニューから行くか、アドレスバーにchrome://extensionsと入力することで遷移できます。

2.ディベロッパーモードをオンにします。そうすると、画像のように「パッケージ化されていない拡張機能を読み込む」ボタンが表示されるので、そこをクリックして「extension」フォルダを選択してください(画像参照)。

3.拡張機能画面に、「Pricetar Tool」が表示されれば成功です。

動作

プライスター画面で、「返金されたものを初期化する」ボタンが追加されます。
このボタンを押すことで処理が始まり、最後のページまで「返金」となっているものを空白で初期化します。
ページが切り替わる際にページ遷移が起こりますが、最後のページまで処理が修了すると、ポップアップが表示され、処理した件数が表示されます。
作業の終了後は、戻るボタンや下のページリンクではなく、ページ上部のメニューから再度戻りたい画面にお戻りください。
